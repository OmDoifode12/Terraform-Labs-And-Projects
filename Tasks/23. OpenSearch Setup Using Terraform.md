# Understanding Amazon OpenSearch Service

Amazon OpenSearch Service is a managed search and analytics service provided by AWS.

It is commonly used for:

* Log analytics
* Application monitoring
* Full-text search
* Security analytics
* Observability platforms

OpenSearch is the successor to Elasticsearch in AWS.

# Step-by-Step Implementation

## Step 1: Navigate to Terraform Directory

```bash id="opes04"
cd /home/bob/terraform
```

### Explanation

This is the Terraform working directory where infrastructure configuration files are stored.

---

## Step 2: Create main.tf

```bash id="opes05"
touch main.tf
```

### Explanation

Terraform resources are defined inside `.tf` files.

---

## Step 3: Open main.tf

```bash id="opes06"
vi main.tf
```

Press:

```text id="opes07"
i
```

to enter INSERT mode.

---

## Step 4: Add Terraform Configuration

```hcl id="opes08"
resource "aws_opensearch_domain" "devops_es" {
  domain_name    = "devops-es"
  engine_version = "OpenSearch_1.0"

  cluster_config {
    instance_type  = "t3.small.search"
    instance_count = 1
  }

  ebs_options {
    ebs_enabled = true
    volume_size = 10
  }
}
```

---

# Code Explanation

| Block                 | Purpose                   |
| --------------------- | ------------------------- |
| aws_opensearch_domain | Creates OpenSearch domain |
| domain_name           | OpenSearch domain name    |
| engine_version        | OpenSearch engine version |
| cluster_config        | Defines cluster settings  |
| instance_type         | OpenSearch instance type  |
| instance_count        | Number of nodes           |
| ebs_options           | Storage configuration     |
| volume_size           | EBS storage size          |

---

# Understanding EBS Storage

```hcl id="opes09"
ebs_enabled = true
```

enables persistent storage for OpenSearch.

```hcl id="opes10"
volume_size = 10
```

allocates:

```text id="opes11"
10 GB
```

of storage.

---

# Understanding OpenSearch Engine Versions

```hcl id="opes12"
engine_version = "OpenSearch_1.0"
```

specifies the OpenSearch engine version.

AWS supports multiple OpenSearch versions for compatibility and feature support.
