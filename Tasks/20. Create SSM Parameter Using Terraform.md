# Day 20 – Create SSM Parameter Using Terraform

---

## Task Overview

As part of AWS infrastructure automation, the Nautilus DevOps team needs to create an AWS Systems Manager (SSM) Parameter using Terraform. AWS SSM Parameter Store is used to securely store configuration data, secrets, environment variables, and application settings.

For this task, create an SSM parameter named `xfusion-ssm-parameter` with value `xfusion-value` using Terraform.

---

# Objective

| Property          | Value                   |
| ----------------- | ----------------------- |
| Resource Type     | SSM Parameter           |
| Parameter Name    | `xfusion-ssm-parameter` |
| Parameter Type    | `String`                |
| Parameter Value   | `xfusion-value`         |
| AWS Region        | `us-east-1`             |
| Provisioning Tool | Terraform               |

---

# Understanding AWS SSM Parameter Store

AWS Systems Manager Parameter Store is a secure storage service for:

* Configuration values
* Secrets
* Environment variables
* Database credentials
* API keys

It helps applications retrieve configuration securely without hardcoding values.

---

# Common Use Cases

| Use Case              |
| --------------------- |
| Application configs   |
| Database passwords    |
| API keys              |
| Environment variables |
| CI/CD secrets         |

---

# Understanding Parameter Types

AWS SSM supports multiple parameter types:

| Type         | Description              |
| ------------ | ------------------------ |
| String       | Plain text values        |
| StringList   | Comma-separated values   |
| SecureString | Encrypted sensitive data |

For this task:

```text id="3v9az1"
String
```

type is used.

---

## Step 3: Open main.tf

```bash id="9cxer3"
vi main.tf
```

Press:

```text id="0dfgt4"

```

to enter INSERT mode.

---

## Step 4: Add Terraform Configuration

```hcl id="1eght5"
resource "aws_ssm_parameter" "xfusion_parameter" {
  name  = "xfusion-ssm-parameter"
  type  = "String"
  value = "xfusion-value"
}
```

---

# Code Explanation

| Block             | Purpose                |
| ----------------- | ---------------------- |
| aws_ssm_parameter | Creates SSM parameter  |
| name              | Parameter name         |
| type              | Parameter type         |
| value             | Stored parameter value |

---

# Understanding Parameter Values
