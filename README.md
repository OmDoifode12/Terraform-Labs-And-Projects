# About This Repository

This repository documents my complete hands-on journey learning and implementing **Terraform on AWS Cloud** through real-world infrastructure automation labs and projects.

The focus of this repository is not just writing Terraform code — but understanding how real production cloud infrastructure is designed, provisioned, secured, automated, modified, and managed using Infrastructure as Code (IaC).

Every project includes:

* Real AWS infrastructure scenarios
* Step-by-step implementation
* Terraform configurations
* AWS CLI verification
* Resource lifecycle management
* Infrastructure automation concepts
* Best practices
* Key learnings
* Interview questions and answers

# Terraform Workflow Used

## 1. Write Infrastructure Code

```bash
main.tf
variables.tf
```

---

## 2. Initialize Terraform

```bash
terraform init
```

---

## 3. Validate Configuration

```bash
terraform validate
```

---

## 4. Preview Infrastructure Changes

```bash
terraform plan
```

---

## 5. Apply Infrastructure

```bash
terraform apply
```

---

## 6. Verify AWS Resources

Using:

* AWS CLI
* Terraform State
* AWS Console

---

## 7. Maintain Infrastructure State

```bash
terraform plan
```

Ensure:

```text
No changes. Your infrastructure matches the configuration.
```
